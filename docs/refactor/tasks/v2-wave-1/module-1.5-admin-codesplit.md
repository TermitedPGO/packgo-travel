## Goal
Convert all 37 admin tab imports in `client/src/pages/Admin.tsx` from eager static `import X from './admin/X'` to `const X = lazy(() => import('./admin/X'))`. Wrap rendered tabs in `<Suspense fallback={<LoadingPage/>}>`. Goal: Admin.js shell drops from ~990 KB to <200 KB; each tab becomes its own 25-60 KB chunk loaded only when navigated to. Mobile-admin TTI improves dramatically (Jeff often opens admin on his phone for quick checks).

## Pre-requisites
- Working tree clean. Husky pre-commit tsc gate active.
- **No dependency on other Wave 1 modules.** Can run parallel to 1.1-1.4.
- **Verify Vite is the bundler** (confirmed: `package.json` build script uses `vite build`). React.lazy + Vite is well-supported with no config change needed.
