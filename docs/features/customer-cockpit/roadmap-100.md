## Phase 1 全渠道進場(+15,最大的一塊)

目標:系統外不存在案子。微信 / iMessage / 電話客人跟 Gmail 客人一樣活在駕駛艙裡。

範圍:
1. 存量進場:把 客人檔案/ 現有 14 個非 Gmail 案子批次建進系統(拖 案件資料.md 或批次腳本;全 draft、售價照 Jeff 核對過的檔、不標付款)。先由 Jeff 手拖 1-2 個驗流程,順了再批次。
2. 增量進場:聊天截圖 / 聊天匯出檔丟進聊天框 → AI 認人(比對既有客人,認不得就問)、建互動紀錄(channel=wechat/sms,標對話真實時間不是歸檔時間)、更新真相條。目前拖檔只會歸檔成文件,要升級成「讀懂並寫進互動時間軸」。
3. 進場紀律(人的部分,Jeff 同意才算數):任何報價出手前,案子必須先在系統裡。

明確不做:接微信個人號、手機 App、自動抓 iMessage(全在憲法非目標)。

驗收:_客人總覽.md 上每一個活案在系統裡都有檔;新開一個微信案從截圖到真相條正確 ≤ 3 分鐘。

決策點(Jeff):批次進場什麼時候跑;進場紀律要不要立為規矩。

## Phase 2 每個數字有出處(+8)

目標:$6,635 那種錯由系統自己抓,不靠人記得對。

範圍:
1. 訂單金額對 invoice:訂單掛了 invoice/確認單 PDF 時,看門狗自動比對 totalPrice 與 PDF 解析出的金額,對不上跳黃卡(兩個數字並排)。沿用既有 attachmentParser + customerDocsText,零新依賴。
2. supplierCost 補登流程:建單/補單時 AI 從供應商文件搬成本(只搬運),margin 看門狗不再對大單瞎;成本永不出現在客人面(既有紅線)。
3. 收款建議(決策點):接既有 Plaid 流水,收到一筆與某訂單金額吻合的入帳 → 出黃卡「疑似 X 單收款,確認就標已收」。AI 只建議,標記永遠 Jeff 點。此項碰錢邊界,Jeff 點頭才做。
4. 補資料六條(不用寫 code,Jeff 聊天框打):ORD-0003 改 6621.40、補 PHX 單、標付款、劉偉國、Jenny 名字、藏 Better way To survive。

驗收:重跑 scorecard,正確性維度回 90+;金額不符案例可被看門狗重現攔截。

## Phase 3 草稿誠實度零容忍(+4)

目標:出自這頁的每一句對客文字,都對得上系統事實。

範圍:
1. 吹牛 gate:草稿宣稱「已寄 / 已提供 / 已確認」的,必須對得上確定性交付事實(customerFacts deriveDelivered),對不上擋下重生(6/29 Leslie 信的斷根)。
2. 抬頭 gate:稱呼必須等於原信寄件人(或不寫名字),不符不落卡。
3. 承諾追蹤:草稿裡答應的事(「週五可取件」)寄出後自動進看門狗承諾清單,過期沒兌現跳卡。
4. 草稿評分工具(既有 eval)改月度自動跑,分數落在 docs,劣化立刻可見。
5. 順帶清舊帳:跟進卡「報價 N 天沒回」字樣改成確定性措辭(系統沒寄過報價就不准寫報價)。

驗收:eval 三宗罪(吹牛 / 重複承諾 / 認錯人)連續兩輪零出現。

## Phase 4 頁面主動排你的一天(+3)

目標:打開 /ops/customers 第一眼是今天該做的事,不用逐個客人翻。

範圍:左欄頂部或首屏一條「今日」帶數字,展開列:到期跟進、報價將過期(14 天效期倒數)、承諾未兌現、出發倒數 T-30/T-7 證件檢查(護照效期、簽證、選位提醒)、尾款到期。全部規則算(讀既有欄位),零 LLM,寧漏勿誤;每項點了直接跳到該客人。

明確不做:排程摘要信、推播(非目標);不做成另一個 dashboard 頁。

決策點(Jeff):放左欄頂還是中欄空狀態(沒選客人時那塊白)。

驗收:Jeff 早上只看這條就知道當天客人面全部待辦;連續一週無誤報。

## Phase 5 學習迴圈閉環(+2)

目標:第十個同類案子比第一個省一半力。

範圍:案子完結(completed/cancelled)觸發回寫:這個客人學到的進他的記憶(已有),這一類案子學到的進共用經驗(供應商雷、路線經驗、定價經驗),晚間批次;AI 接同類新案時自動帶出「上次這類案子的三個教訓」。與桌面 工作筆記/skill 迴圈對齊([[feedback_packgo_learning_loop]])。

驗收:新開一個阿拉斯加郵輪案,AI 第一回合就主動引用陳案/美玲案的經驗。

## Phase 6 單一入口 + 自我體檢(+3)

目標:全公司客人操作只有這一頁;它自己告訴你它幾分。

範圍:

3. 雜訊清源:建檔自動帶 Gmail 顯示名、自家信箱黑名單(jeffhsieh09/0909)不建客人。

驗收:一個月內 Jeff 沒有開過 /workspace 客人頁;月度 scorecard 自動出分。
