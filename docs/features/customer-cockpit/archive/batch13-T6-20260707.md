## 1(P2)附件自動掛跟措辭脫鉤 — `6856c45`

批十二把 re-attach 綁在 detectAttachmentClaim(文案聲稱「附上」才掛),但工具指引又教「檔案
沒掛上前別寫附上」→ 互鎖成老實草稿(沒寫附上)永遠不觸發掛、拿不到附件。修:抽出
`fetchRecentGeneratedAttachments(db, profileId, now, window)` —— 簽名不吃 body,脫鉤是結構性的。
produceFollowupDraftForProfile 建草稿時無條件呼叫:只要 30 分內有 generated 文件就掛最新一份,
無論文案措辭。誠實閘方向不變:文案聲稱附上但這裡沒撈到(hasAttachment=false)照樣擋。
測試 followupDraftOnDemand.test.ts(mock db):撈最新一份、只掛一份、超窗不掛、無文件/查失敗回 []。

## 4(P3)聊天送出鈕串流時位移 — `bf77cb9`

根因是 flexbox:flex-1 訊息捲動區沒有 min-h-0,串流時內容長高把捲動區撐大、將下面的輸入列頂出
容器(父層 overflow-hidden 裁掉),送出鈕位移點不中。修:訊息區加 min-h-0(真的內部捲動)+
輸入列加 flex-shrink-0(釘在底部)。純 CSS class,實機串流視覺為 ship 後驗收。
