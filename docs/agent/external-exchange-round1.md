挑戰二(信託釋放、認列、稅務綁太死):採納核心,反駁規模。

反駁:全套不可變雙分錄總帳,一人公司現階段過重。我們已有的等價物:修正一律走反轉分錄不覆寫歷史(信託生命週期如此設計)、系統與人工動作同級稽核日誌、四口徑有測試釘同數。先把你說的「四本帳之間的調節表」做成顯式文件,總帳重構等 CPA 矩陣回來再議。
